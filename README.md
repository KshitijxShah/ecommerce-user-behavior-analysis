# 📊 E-commerce User Behavior & Funnel Analysis

## 📌 Overview

This project analyzes user behavior on an e-commerce platform to understand how users move through the purchase funnel (View → Cart → Purchase).

The goal is to identify drop-off points, analyze product/category performance, and understand user engagement using multiple tools.

---

## 🎯 Objective

* Analyze user journey from product view to purchase
* Identify where users drop off in the funnel
* Understand which categories and users drive conversions

---

## 🛠️ Tools Used

* **Excel** → Data cleaning, pivot tables, initial analysis
* **SQL** → Querying data and answering business questions
* **Power BI** → Interactive dashboard and visualization

## 📊 Analysis Performed

### 🔹 Funnel Analysis

* Measured number of users at each stage:

  * View
  * Cart
  * Purchase
* Calculated conversion rates between stages

---

### 🔹 Category Analysis

* Identified top-performing categories based on purchases
* Compared user engagement across categories

---

### 🔹 User Analysis

* Found most active users
* Identified users who completed purchases
* Compared active users vs converting users

---

### 🔹 Trend Analysis

* Analyzed daily user activity
* Observed patterns in engagement

---

## 📈 Key Insights

* Significant drop-off observed from **View → Cart stage**
* Conversion from **Cart → Purchase is relatively higher**, indicating strong buying intent
* **Electronics category drives the majority of purchases**
* Only a small percentage of users complete the full purchase journey
* A few users contribute disproportionately to total purchases (high-value users)
